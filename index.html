Dativ

How Dativ Works

Dativ is 3rd case in german, and in the most simplest sense, it applies when something is being done with something else.

Whereas in Akkusativ, where something is being done to an object (Direct Object), Dativ applies when something is done with something else (Indirect Object).

This concept doesn't exist in English, so let's consider an example:

I am giving you a present

In this example, the present is the direct object, as it is what is being done too (it is being gifted). However, you is the indirect object, as the gift is given to you (but the verb does not directly affect you). Therefore, it is being done with you.

We can apply this same logic to another English example:

I am coming with you

In this sentence, you is the indirect object as nothing is being done directly to it, it is simply being done with (As you are coming - but with someone).

Practical Knowledge

Because Dativ is invoked when something is done with something else, this means that many prepositions in german force the dativ case:

mit, bei, aus, nach, zu, von, seit all force you to use dativ, meaning that any noun following these prepositions has to be dativ. This is important to remember because the majority of the time you are using dativ, you are probably going to be using these prepositions.

Therefore if you ever want to say your doing something with someone/something, then you automatically know it's dativ.

Contractions

German has several contractions that are specific to dativ. It is impossible to go through everyday life without noticing them:

  • zu dem = zum
  • zu der = zur
  • bei dem = beim
  • von dem = vom
  • in dem = im

This is also just a good basic thing to know.

Von (and aus)

Von is incredibly important to know, and I feel like many of you guys aren't sure of how to use it.

Aus means from, but only in a geographical context, so if you are (for example) from a country/state you can use it.

Von is what you use for any other situation where you want to say from. Additionally, because german doesn't have an exact word for "of", we can use von- for example:

a friend of mine - ein Freund von mir

With Verbs

Dativ Only

Specific verbs are only dativ, or they require both dativ and akkusativ nouns.

For example, glauben (to believe) or helfen (to help) are dativ only verbs. For example:

Ich glaube dem Mann or Ich helfe dir

Dativ + Akkusativ Verbs

Many common verbs require a dativ and akkusativ noun.

For example, schulden (to owe) or schicken (to send), require a dativ indirect object and a akkusativ direct object

For example:

Ich schulde dir etwas
